FAQ
常见问题
快速解答你关心的问题，如有更多疑问请联系客服。

QuantAI 适合新手吗？

不需要专业背景。QuantAI 面向所有投资者设计——从零基础到专业量化均可使用。新手可从 AI 顾问的建议出发，专业用户可自定义策略参数。

行情数据是实时的吗？

是的。加密货币通过 Binance WebSocket 实时推送；传统资产（外汇/贵金属/指数）通过 Yahoo Finance 定时轮询获取。

自动交易安全吗？

只有在你主动开启「AI 自动下单」开关后才执行，默认所有交易需手动确认。配合单日最大亏损、持仓比例限制、夜间保护模式等多重风控。

支持哪些支付方式？

支持信用卡（Visa/MasterCard）、PayPal 和加密货币（BTC/ETH/USDT）。7天内未使用付费功能可申请退款。加密货币支付点击"₿ 加密货币"按钮即可完成。

复制交易有风险吗？

有。建议设置「单笔最大金额」限制风险、开启「日亏损保护」，并分散跟单多个信号源，不要把全部资金投入单一交易者。

API Key 安全吗？

API Key 仅存储在本地浏览器 localStorage，不上传服务器。建议开启 IP 白名单，只开「读取+交易」权限，不开「提现」权限。
