## 🏢 产品概述

**QuantAI** 是一款多资产智能量化交易平台，支持 Web + PWA 安装，提供：

- **复制交易**（跟随顶级交易者实时同步）
- **自动建仓**（Elite 专属：策略触发 / 定时定投 / 跟单同步）

- **信号广播**（实时交易信号发布与订阅）
- **宏观数据**（FRED 美联储：利率 / CPI / 非农 / GDP）

## ❓ 常见问题 FAQ

### 📌 基础使用

**Q1：QuantAI 是什么？我需要是专业交易员才能使用吗？**
A：不需要。QuantAI 面向所有投资者设计——从零基础到专业量化均可使用。新手可以从 AI 顾问的建议出发，专业用户可以自定义策略参数和 Pine Script 代码。

**Q2：QuantAI 可以在哪些设备上使用？**
A：支持所有现代浏览器（Chrome / Safari / Edge / Firefox）。还支持 PWA 安装，可像 App 一样添加到手机桌面使用，体验媲美原生 App。

**Q3：行情数据是实时的吗？**
A：是的。加密货币行情通过 Binance WebSocket 实时推送（1分钟更新）；传统资产（外汇/贵金属/指数）通过 Yahoo Finance 定时轮询获取。数据来源标注在K线图下方。

**Q4：如何查看特定品种的K线图？**
A：进入「行情」页面 → 点击任意品种右侧的「查看」按钮，或直接在「仪表盘」主界面查看 BTC/USDT K线（可切换品种和时间周期：1分/5分/15分/1小时/4小时/1日）。

**Q5：宏观数据（FRED）是什么？为什么显示 offline？**
A：FRED 宏观数据来自美联储圣路易斯联储，包括联邦基准利率、10年/2年期国债收益率、CPI、核心PCE、非农就业、失业率等关键经济指标。如果显示"offline · static data"，说明当前网络无法访问 FRED API（常见于中国大陆网络），此时显示上次已知的静态默认值，不影响正常使用。

### 🤖 AI 顾问

**Q6：AI 顾问能做什么？**
A：AI 顾问可以：
- 分析市场行情、给出多空判断
- 帮你计算仓位分配（告诉 AI 你的资金规模）
- 推荐适合当前行情的策略
- 解读技术指标（RSI、MACD、EMA等）
- 回答关于平台功能的问题

**Q7：AI 顾问的分析准确吗？**
A：AI 顾问提供的是参考性分析，不构成投资建议。市场存在不确定性，所有交易决策请自行判断，做好止损管理。

**Q8：如何让 AI 顾问分析我的持仓？**
A：进入「持仓」页面 → 点击持仓旁的「AI分析」按钮，系统会自动跳转到 AI 顾问并发起分析对话。

### ⚡ 策略管理

**Q9：策略回测是真实数据还是模拟数据？**
A：回测使用历史市场数据进行模拟，结果反映策略在过去行情下的表现。过往绩效不代表未来收益，请结合风控使用。

**Q10：Basic 套餐只能创建 2 个策略，可以删除旧的创建新的吗？**
A：可以。在「策略管理」页面点击策略右侧的「停止」→「删除」后，就可以创建新策略。删除后数据无法恢复，请确认后操作。

**Q11：如何使用自动建仓（Auto Open）？**
A：自动建仓为 Elite 专属功能，需升级到 $199/月套餐。提供三种模式：
1. **策略触发**：RSI/MACD/EMA 等技术指标满足条件时自动开仓
2. **定时定投**：按小时/天/周/月自动分批买入（DCA）
3. **跟单同步**：绑定信号源，自动复制其每笔开仓操作

### 🔁 复制交易

**Q12：复制交易是如何工作的？**
A：在「复制交易」页面选择交易者 → 点击「跟单」→ 设置跟单比例和最大金额。平台会实时同步该交易者的信号，在你的账户中按比例执行。

**Q13：跟单有没有风险？**
A：有。即便是排行榜第一的交易者也存在亏损风险。建议：
- 设置「单笔最大金额」限制单次风险
- 开启「日亏损保护」自动停损
- 不要把全部资金投入单一信号源

**Q14：怎么成为信号源（被别人跟单）？**
A：进入「信号广播」页面 → 填写交易对、方向、入场价、止损止盈 → 点击「广播信号」。其他用户可以订阅你的信号，并在「自动建仓-跟单同步」中绑定你作为信号源。

### 💳 账户与支付

**Q15：如何连接券商账户？**
A：进入「我的账户」→「已连接券商」→ 点击「添加新券商」→ 选择对应券商（币安/OKX/MT5等）→ 填写 API Key 和 Secret → 点击「验证并连接」。

**Q16：我的 API Key 安全吗？**
A：API Key 仅存储在你的本地浏览器 localStorage 中，不会上传到服务器。我们强烈建议你在券商设置 API 时：①开启 IP 白名单 ②只开「读取+交易」权限，不开「提现」权限。

**Q17：如何取消订阅/降级套餐？**
A：请联系客服处理。降级生效时间为当前计费周期结束后，期间仍可使用当前套餐权益。

**Q18：支持哪些支付方式？**
A：目前支持信用卡（Visa/MasterCard）、PayPal。加密货币支付请联系客服。

### 🔧 故障排查

**Q19：K线图加载不出来 / 显示空白，怎么办？**
A：
1. 检查网络是否正常
2. 刷新页面（按 Ctrl+Shift+R 强制刷新，清除缓存）
3. 换一个浏览器尝试
4. 检查是否有浏览器插件（如 AdBlock）拦截了数据请求
5. 如仍无法解决，请截图联系客服

**Q20：行情数据不更新了（停止跳动），怎么处理？**
A：右上角会显示"⚠️ 网络已断开"提示。通常刷新页面即可恢复。加密货币行情依赖 Binance WebSocket，如果 Binance 服务器有波动会临时断线，一般几分钟内自动重连。

**Q21：FRED 宏观数据一直显示 "…"（加载中），怎么回事？**
A：FRED API 需要访问 `api.stlouisfed.org`，在中国大陆网络通常无法直连。平台有 CORS 代理降级处理，但代理也可能不稳定。最终兜底会显示静态默认值（标注"last known"），不影响平台其他功能使用。

**Q22：回测运行后结果全是 0，这正常吗？**
A：如果参数设置不合理（如日期范围太短、仓位比例过低）可能出现无效结果。建议：日期范围至少选 90 天，仓位比例选 10%-20%，初始资金至少 $1,000。

**Q23：语言切换后部分文字没有更新，如何处理？**
A：刷新页面即可（Ctrl+R）。语言设置已保存，刷新后会完整应用新语言。

**Q24：在手机上用，侧边栏怎么打开？**
A：手机上左上角有「☰」汉堡菜单按钮，点击即可展开导航侧栏。PWA 安装后体验更佳（全屏、无浏览器标题栏）。

### 📊 策略市场

**Q25：策略市场上的策略是谁创建的？可信吗？**
A：策略由平台用户发布，均经过审核后上架。每个策略展示历史回测数据（月均收益、胜率、回撤、总交易次数），建议参考这些数据评估策略质量，并先用小仓位测试。

**Q26：如何发布我自己的策略？**
A：进入「策略市场」→ 点击「📤 发布策略」→ 填写策略名称、适用品种、定价、简介、Pine Script/伪代码 → 提交审核。审核通过后自动上架（通常 1-3 个工作日）。

**Q27：购买的策略如何使用？**
A：点击策略卡片上的「复制」按钮，策略会复制到你的「策略管理」页面，然后配置参数并启动即可。

### 🔐 隐私与安全

**Q28：QuantAI 会不会替我自动下单？**
A：只有在你主动开启「AI 自动下单」开关（在「账户」→「风控设置」中）后，AI 才会自动执行。默认状态下所有交易需要手动确认。

**Q29：如何设置风控保护？**
A：进入「我的账户」→「风控设置」：
- **单日最大亏损**：超过设定值后自动停止所有策略
- **最大持仓比例**：单品种不超过总资产的设定百分比
- **夜间保护模式**：22:00-07:00 暂停所有自动交易

## 📋 客服 Bot 回复规范

> 以下为 Bot 回复指南，人工客服也请参考。

1. **先理解，再回答**：确认用户描述的具体症状，不要猜测
2. **引导至功能**：尽量告诉用户在哪个页面/按钮可以解决
3. **升级话术**：遇到免费版限制时，温和引导升级，不强推
4. **转人工条件**：涉及支付、账号安全、数据异常的问题，直接转人工
5. **语言匹配**：用户用什么语言发来就用什么语言回复
